18.1 Assignment

Solidify Terminology

  • What is HTTP? - Hyper Text Transfer Protocol - how browsers and servers communicate
  • What is a URL? - Universal Resource Locator - it's an address for some internet resource
  • What is DNS? - Domain Name System - takes user-readable URLs and converts them into IP addresses
  • What is a query string? - it allows you to pass key-value pairs into the URL
  • What are two HTTP verbs and how are they different? - POST - sends data to the server - GET - retreives data from the server
  • What is an HTTP request? - a request from a user to a server which follows the HTTP protocol
  • What is an HTTP response? - a response from the server to the user
  • What is an HTTP header? - it provides additional infrmation about the request/ responce Examples: Request Header and Responce Header
  • What are the processes that happen when you type“http://somesite.com/some/page.html” into a browser?
    1. The browser converts the name into an IP address with DNS
    2. The browser then makes a request to that IP address
    3. The server will send a responce
    4. The browser takes the responce and makes a DOM. For additional needed resourcs, The browser makes a seperate request and receives the server's responce for each
